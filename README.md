## 🚀 Getting Started

### Prerequisites
To compile and run these codes, you need a C++ compiler installed on your system (e.g., MinGW for Windows or GCC for Linux/Mac).

### How to Run
1. Clone this repository:
   git clone [https://github.com/AnishPrakash/My-C-Portfolio.git](https://github.com/AnishPrakash/My-C-Portfolio.git)
2. Compile a File:
   g++ filename.cpp -o output
3. Run the output:
   Windows: output.exe
   Mac/Linux: ./output
